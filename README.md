# Daksh Mehta - Personal Branding Website

A modern, dark-themed personal branding website for Daksh Mehta, CTO at Romin Interactive and Laravel/ERPNext Consultant.

## Features

- **Single Page Design**: All content organized in a smooth-scrolling single page
- **Dark Theme**: Professional dark mode with teal accent colors
- **Responsive**: Works perfectly on desktop, tablet, and mobile devices
- **Modern Design**: Clean, minimal, and professional appearance
- **Sections Included**:
  - Hero/About section with professional introduction
  - Skills & Expertise showcase
  - Past Projects portfolio
  - Client Testimonials
  - Blog posts preview
  - Contact form
  - Social media links

## Technologies Used

- Pure HTML5, CSS3, and JavaScript
- No external dependencies
- Optimized for GitHub Pages hosting

## Hosting on GitHub Pages

1. Create a new repository on GitHub
2. Upload the `index.html` file to the repository
3. Go to repository Settings > Pages
4. Select "Deploy from a branch" and choose "main" branch
5. Your website will be available at `https://yourusername.github.io/repository-name`

## Customization

### Colors
The website uses a dark theme with these main colors:
- Background: `#0a0a0a` (dark black)
- Cards/Sections: `#1a1a1a` (dark gray)
- Accent: `#00d4aa` (teal green)
- Text: `#e0e0e0` (light gray)

### Content Updates
To update content, simply edit the HTML file:
- Personal information in the hero section
- Skills in the skills section
- Projects in the projects section
- Testimonials in the testimonials section
- Blog posts in the blog section

### Profile Image
Replace the placeholder image URL in the hero section with your actual photo:
```html
<img src="your-photo-url.jpg" alt="Daksh Mehta">
```

### Social Links
Update the social media links in the contact section with your actual profiles.
